# Ruggito Feed

Feed RSS personale per il podcast **Il Ruggito del Coniglio** di Rai Radio 2.

## Descrizione

Questo repository contiene il feed RSS aggiornato automaticamente ogni giorno feriale con i nuovi episodi de Il Ruggito del Coniglio, il programma cult di Rai Radio 2 condotto da Marco Presta e Antonello Dose.

## Feed URL

```
https://raw.githubusercontent.com/MonsterKingu/ruggito-feed/master/feed_ilruggitodelconiglio.xml
```

## Aggiornamento automatico

Il feed viene aggiornato automaticamente ogni giorno feriale alle 12:00 ora italiana tramite GitHub Actions, scaricando i nuovi episodi dal sito RaiPlay Sound.

## Sorgente

Gli episodi vengono scaricati dal feed originale di [giuliomagnifico](https://github.com/giuliomagnifico/raiplay-feed).

## Utilizzo

Questo feed viene utilizzato dall'app **Ruggito** — un'app Android dedicata alla riproduzione de Il Ruggito del Coniglio.

## Licenza

I contenuti audio appartengono a Rai Radio 2. Questo repository gestisce solo i metadati del feed RSS.
